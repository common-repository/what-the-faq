== Description ==

Create and configure Frequently Asked Questions using a simple, clean interface. Easily tweak the design with a handful of options at your disposal, while still inheriting the look and feel of your existing theme.

== Installation ==

Install, activate, done.

== How To Use ==

* Go to **Settings** -> **What The FAQ** to initially set up things the way you want.
* Start creating FAQs by clicking **What the FAQ** -> **Add New**.
* Optional: **<a href="https://rocketapps.com.au/product/what-the-faq-pro/?origin=wtf-wordpress-page">Upgrade to Pro</a>** for more features and control.

== Uninstall ==

Deactivate the plugin, delete if desired. Deactivating does not delete existing FAQs from the database, so if you reactivate the plug-in later all your FAQs will still be there.
